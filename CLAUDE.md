# Claude Code ルール

## Git操作の厳守事項

### 絶対禁止
- **mainブランチで直接作業しない**
- **ユーザーの明示的な指示なしにコミット・マージ・プッシュしない**

### 作業開始時の確認
1. 現在のブランチを確認する（`git branch`）
2. mainブランチにいる場合は、必ずユーザーに確認：「新しいfeatureブランチを作成しますか？」
3. ブランチ名はユーザーに確認してから決定する

### コミット・マージ・プッシュ
- **コミット**: ユーザーが「コミットして」と言うまで行わない
- **マージ**: ユーザーが「マージして」と言うまで行わない
- **プッシュ**: ユーザーが「プッシュして」と言うまで行わない
- 複合指示（「コミットしてプッシュして」など）以外は、各操作を個別に確認する

### ブランチ切り替え
- ユーザーの指示なしにブランチを切り替えない
- マージ完了後も、次の作業について必ず確認する

## 言語
- コミットメッセージは日本語で書く
